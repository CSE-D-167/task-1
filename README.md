# Personal Portfolio Website

## Technologies

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Installation

Install dependencies

```bash
npm install
```

Import

```
portfolio.sql
```

into MySQL.

Run

```bash
node server.js
```

Open

```
index.html
```

in browser.

Server

```
http://localhost:3000
```

API

```
GET /projects
```

## Features

- Responsive Portfolio
- About Section
- Skills Section
- Dynamic Projects
- Contact Information
- MySQL Database
- REST API
